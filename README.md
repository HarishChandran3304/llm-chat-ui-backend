# llm-chat-ui-backend

## Setup:

### 1) Create a virtual environment

```shell
python3 -m venv venv
```

### 2) Activate the virtual environment

For Linux:

```shell
source ./venv/bin/activate
```

For Windows:

```shell
. venv/bin/
```

### 3) Install dependencies

```shell
pip install -r requirements.txt
```

## Run Development Server:

```shell
fastapi dev main.py
```
